# Flutter-Front - Dasom Community App

경희대학교 컴퓨터 공학과 과동아리 Dasom을 위한 커뮤니티성 앱 프로젝트.
http://dasom.io/ 기존에 있던 다솜 서버를 사용해서 기본적인 데이터를 받아왔다. 추가적으로 필요하다고 생각했던 기능은 따로 서버를 추가하여 추가적으로 구현하였다.

다음은 사용한 기술스택 정리이다.
